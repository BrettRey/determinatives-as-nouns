# Objection matrix — falsification conditions for the central claim

Planning asset for §1 (preview), §4 (steelman) and §5 (Table 2). Two tables: (i) what would falsify the proposal (Table A; planning aid only, not in paper); (ii) compressed comparison with nearby proposals (Table B; becomes Table 2 in §5).

## Table A: Falsification conditions

| Objection | Falsification condition | Reply | Where in paper |
|-----------|------------------------|-------|----------------|
| Articles never stand alone | If no defective members exist elsewhere in Noun, sub-category membership requires standing alone | Membership criterion (§3) is at the sub-category profile level, not the lexeme level; lexically defective members admitted on paradigmatic grounds. Defective proper nouns (`\mention{*the John}`, `\mention{the Hague}`); CGEL 371-2 already marks `\mention{the/a}` as lexically defective; PHP 2007: 22-23 do the same for `\mention{every}`, `\mention{no}` | §1 (seed), §3 (criterion), §4 (full) |
| Fused-head productivity is overstated within the determinative class | If fused-head use turns out to be lexically restricted within determinatives (only a few members admit it), making the determinative pattern look like the adjective Mod-Head pattern | The defective set (`\mention{the, a, every, no}`) is small and independently flagged in CGEL/PHP; the broad set admitting fused-head (`\mention{some, all, both, many, few, several, most, this/these, that/those, each, either, neither, much, enough}`) covers most of the class. The pattern is broadly productive, not lexically restricted | §3.1 |
| Mod-Head fusion is more productive than the discriminator allows | If adjective Mod-Head fusion turns out to be productive across the adjective class (more than the lexicalised set `\mention{the rich, the poor, the elderly, ...}`) | The Mod-Head set is well-documented as a small lexicalised class in CGEL §5.9 and PHP 2007; productive Mod-Head fusion would be a separate empirical claim that the literature doesn't support | §3.1 |
| Structural saturation asymmetry doesn't hold | If simple fused-head NPs require a covert determiner or other unstated structure, or if Mod-Head fusion can occur without an overt determiner in the NP | Simple fused-head NPs are full NPs with no further structure (`\mention{some left}`, `\mention{many came}`); Mod-Head fusion always requires an overt determiner (`\mention{the rich}` OK, `\mention{*rich}` alone as class-NP isn't). The asymmetry is observable | §3.1 |
| Determinatives don't show noun morphology (no plural, no genitive in most cases) | If supercategory membership requires shared inflectional morphology | The membership criterion (§3) is functional, not morphological: items satisfy the criterion if they canonically head NPs in subject, object, or PrepC functions. Pronouns already fail morphological tests common nouns satisfy and are still admitted to Noun | §3 |
| Determinatives take AdvP modifiers, common nouns take AdjP — categorial difference | If sub-cluster combinatorics can't be modelled within a single supercategory | Sub-cluster combinatorics is standard practice (PHP 2010: 13-14 already alternate `\mention{few}` between Det and Mod functions without changing category) | §6 |
| The unification has no empirical payoff; just renaming | If neither fused-head reanalysis nor NP:Det uniformity yields real simplification | §3 demonstrates both: fusion-of-functions apparatus drops out; the disjunctive Det filler collapses to NP | §3 |
| The supercategory becomes vacuous if it admits all four sub-categories | If the supercategory predicts nothing beyond what each sub-category projects on its own | The supercategory predicts that all four sub-categories project to NP and head NPs in canonical nominal functions. For three sub-categories this is direct (CGEL); for determinatives it follows from the §3.1 reanalysis. The supercategory's content is the unified projection-and-distribution generalisation | §3 (criterion + §3.1) |
| The same proposal has been made before, possibly in stronger form | If Spinillo, Van Eynde, or Anderson already commit to the fourfold-Noun architecture | They don't: Spinillo stops at the negative case; Van Eynde splits between A and N; Anderson groups three under {N} and excludes common N | §5 |
| Cross-linguistic data refute the move | If languages without articles or with rich nominal agreement show that determinatives pattern as a separate category | Out of scope; English-only paper. Cross-linguistic projection is future work | §1 (scope), §7 |
