# Lean Formalization Supplement

This directory contains the companion Lean 4 formalization for
_Determinatives as nouns in English_.

Scope:

- fourfold Noun superordinate
- field-relative projectibility profiles
- noun-headed determiner uniformity
- category/function overlap without collapse

Build from this directory with:

```bash
lake build RequestProject
```

Main file:

- `RequestProject/CGELBankFourfoldNoun.lean`

This is a narrow witness model, not a data-linked formalization of the full
CGELBank annotation scheme.
